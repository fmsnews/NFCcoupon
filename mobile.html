Mobile Application

รูปที่ 1.2 (Some snapshots of WingBonus Mobile Application(a)

เพื่ออำนวยความสะดวกให้กับผู้ใช้และสำหรับการทำงานที่เหมาะสมของระบบได้พัฒนาแอพพลิเคชั่น Androidเมื่อผู้ใช้เรียกใช้แอพพลิเคชั่นเป็นครั้งแรกหน้าจอจะปรากฏดังรูปที่ 1.2(a)

รูปที่ 1.3 Some snapshots of WingBonus Mobile Application(b)

ผู้ใช้ที่มีการลงทะเบียนควรป้อนข้อมูลประจำตัวของเขาในหน้าจอการตั้งค่าของระบบ หน้าจอนี้สามารถเข้าถึงได้โดยการเลือกเมนู นอกจากนี้ยังกำหนดวิธีดูคูปองได้หลายแบบ เช่นแสดงคูปองเรียงลำดับตามวันที่หรือตามชื่อ เรื่องตามจำนวนของคูปองที่แสดง เรื่องตามประเภทที่ชื่นชอบและอื่นๆเมื่อผู้ใช้มีการระบุ(ผู้ใช้ที่ลงทะเบียนหรือไม่ประสงค์ออกนาม)แอพพลิเคชั่นจะแสดงเมนูหลักตัวเลือกทั้งหมดสามารถเข้าถึงได้ผ่านทางเมนูหลักที่แสดงในรูปนี้

รูปที่ 1.4 Some snapshots of WingBonus Mobile Application(c)

ความแตกต่างของสองตัวเลือก(เมนู Coupons และเมนู Chits) ถูกแยกออกจากกันในการจัดการคูปองและ ChitsจากเมนูCouponsผู้ใช้สามารถดําเนินการใดที่เกี่ยวข้องกับองค์ประกอบเหล่านั้นตัวอย่างเช่นผู้ใช้สามารถดาวน์โหลดคูปองโดยตรงตาม แสดงในรูป

รูปที่ 1.5 Some snapshots of WingBonus Mobile Application(d)

ตัวเลือกการจัดการทั้งหมด เช่น ลบคูปองดูรายละเอียดคูปองแสดงในรูปที่ 1.5 และการใช้คูปองจะสามารถเข้าถึงได้จากเมนูนี้

รูปที่ 1.6 Some snapshots of WingBonus Mobile Application(d)

การจัดเตรียมคูปองใน mobile applicationมีการดำเนินการโดยขั้นตอนการรักษาความปลอดภัยขั้นตอนนี้จะอยู่ในความดูแลของa) การส่งข้อมูลทั้งหมดในฐานข้อมูลโทรศัพท์มือถือเพื่อการตรวจสอบที่ฝั่งเซิร์ฟเวอร์b)เพื่อรับข้อมูลใหม่ทั้งหมดของคูปองจากเซิร์ฟเวอร์และจัดเก็บไว้ในฐานข้อมูลมือถือดังนั้นในขั้นตอนนี้ฝั่งเซิร์ฟเวอร์จะส่งคูปองทั้งหมดตามที่ผู้ใช้ทำเครื่องหมายไว้ก่อนหน้านี้ รวมทั้งคูปองอื่นๆที่ผู้ใช้ได้เลือกจากสมาร์ทโพสต์เตอร์,Bluetoothserversหรือใช้แอพพลิเคชันกับเพื่อนๆไปยังแอพพลิเคชั่นบนมือถือ ของผู้ใช้ผู้ใช้สามารถเข้าถึงรายละเอียดข้อมูลของแต่ละคูปองดังแสดงในรูปที่ 1.6 (d)

รูปที่ 1.7 Some snapshots of WingBonus Mobile Application (e)

จะมีคำอธิบายข้อมูลทางเศรษฐกิจและสถานะของคูปองที่มีการแลกเปลี่ยนก่อนหน้านี้ได้นอกจากนี้ผู้ใช้สามารถขอแผนที่ที่มีข้อมูลเกี่ยวกับสถานที่ที่คูปองสามารถแลกเปลี่ยนได้ ดังแสดงในรูปที่ 1.7(e)

รูปที่ 1.8 Some snapshots of WingBonus Mobile Application(f)

เพื่อจุดประสงค์นี้ WingBonusใช้ API ของ Google Maps ที่ช่วยให้การแสดงเส้นทางจากตำแหน่งปัจจุบันของผู้ใช้ที่ได้รับจาก GPS ถ้ามันมีอยู่ สถานที่ที่ใกล้เคียงที่สุดที่อนุญาตให้คูปองสามารถแลกเปลี่ยนได้ แสดงในรูป (รูปที่ 1.8 (f))
